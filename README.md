# 🛒 MinDo'kon — Onlayn Do'kon

HTML, CSS va Vanilla JavaScript'da yaratilgan oddiy onlayn do'kon loyihasi.

## ✨ Funksiyalar

- Mahsulotlar ro'yxati (kategoriya bo'yicha filtr: Elektronika, Kiyim, Uy jihozlari)
- Savatga mahsulot qo'shish / miqdorini o'zgartirish / o'chirish
- Savat ma'lumotlari `localStorage`'da saqlanadi (sahifani yopib qayta ochsangiz ham saqlanib qoladi)
- Umumiy summani avtomatik hisoblash
- Responsiv dizayn (mobil qurilmalarda ham ishlaydi)

## 📁 Fayllar tuzilishi

```
online-dokon/
├── index.html      # Asosiy sahifa
├── style.css        # Dizayn
├── script.js        # Funksionallik (savat, filtr)
├── products.js       # Mahsulotlar ma'lumotlari
└── README.md         # Ushbu fayl
```

## 🚀 Qanday ishlatish

Shunchaki `index.html` faylini brauzerda oching — server kerak emas.

## 📤 GitHub'ga joylash

1. GitHub'da yangi repository yarating (masalan: `online-dokon`), README qo'shmasdan.
2. Ushbu papkani kompyuteringizga yuklab oling va terminalda o'sha papkaga o'ting.
3. Quyidagi buyruqlarni ketma-ket bajaring:

```bash
git init
git add .
git commit -m "Birinchi commit: Online do'kon loyihasi"
git branch -M main
git remote add origin https://github.com/FOYDALANUVCHI_NOMI/online-dokon.git
git push -u origin main
```

`FOYDALANUVCHI_NOMI` o'rniga o'z GitHub username'ingizni yozing.

## 🌐 GitHub Pages orqali onlayn joylash (bepul)

1. Repository sahifasida **Settings → Pages** bo'limiga o'ting.
2. **Branch**: `main`, **Folder**: `/ (root)` tanlang va **Save** bosing.
3. Bir necha daqiqadan so'ng sayt `https://FOYDALANUVCHI_NOMI.github.io/online-dokon/` manzilida ochiladi.
